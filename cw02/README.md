## Zadanie 1 - Porównanie wydajności systemowych i bibliotecznych funkcji we/wy
* Celem zadania jest napisanie programu porównującego wydajność systemowych i bibliotecznych funkcji wejścia/wyjścia. Program operował będzie na przechowywanej w pliku tablicy rekordów. Dla uproszczenia pojedynczy rekord będzie tablicą bajtów o stałej wielkości. Nazwa pliku, wielkość oraz liczba rekordów stanowić będą argumenty wywołania programu.
